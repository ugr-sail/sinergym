Constructor. Variables, meters and actuators are required to update building model scheme.

Parameters:

  • env_name (str) ‚Äì Name of the environment, required for Sinergym output management.

  • json_file (str) ‚Äì Json file name, path is calculated by the constructor.

  • weather_files (List[str]) ‚Äì List of the weather file names, one of them will be select randomly, path will be calculated by the constructor.

  • variables (Dict[str, Tuple[str, str]]) ‚Äì Specification for EnergyPlus Output:Variable. The key name is custom, then tuple must be the original variable name and the output variable key.

  • meters (Dict[str, str]) ‚Äì Specification for EnergyPlus Output:Meter. The key name is custom, then value is the original EnergyPlus Meters name.

  • actuators (Dict[str, Tuple[str, str, str]]) ‚Äì Specification for EnergyPlus Input Actuators. The key name is custom, then value is a tuple with actuator type, value type and original actuator name.

  • max_ep_store (int) ‚Äì Number of episodes directories will be stored in experiment_path.

  • extra_config (Dict[str, Any]) ‚Äì Dict config with extra configuration which is required to modify building model (may be None).
