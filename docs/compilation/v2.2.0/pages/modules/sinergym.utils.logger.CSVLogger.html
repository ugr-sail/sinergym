class sinergym.utils.logger.CSVLogger(monitor_header: str, progress_header: str, log_progress_file: str, log_file: str | None = None, flag: bool = True)ÔÉÅ

CSV Logger for agent interaction with environment.

Parameters:

  • monitor_header ‚Äì CSV header for sub_run_N/monitor.csv which record interaction step by step.

  • progress_header ‚Äì CSV header for res_N/progress.csv which record main data episode by episode.

  • log_file ‚Äì log_file path for monitor.csv, there will be one CSV per episode.

  • log_progress_file ‚Äì log_file path for progress.csv, there will be only one CSV per whole simulation.

  • flag ‚Äì This flag is used to activate (True) or deactivate (False) Logger in real time.

  • etc (steps_data, rewards, powers,) ‚Äì These arrays are used to record steps data to elaborate main data for progress.csv later.

  • total_timesteps ‚Äì Current episode timesteps executed.

  • total_time_elapsed ‚Äì Current episode time elapsed (simulation seconds).

  • comfort_violation_timesteps ‚Äì Current episode timesteps whose comfort_penalty!=0.

  • steps_data ‚Äì It is a array of str‚Äôs. Each element belong to a step data.
