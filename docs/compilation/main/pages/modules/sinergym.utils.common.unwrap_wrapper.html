sinergym.utils.common.unwrap_wrapper

sinergym.utils.common.unwrap_wrapper(env: Env, wrapper_class: Type[Wrapper]) → Wrapper | None

Retrieve a wrapper object by recursively searching.

Parameters:

  • env (gym.Env) – Environment to unwrap

  • wrapper_class (Type[gym.Wrapper]) – Wrapper to look for

Returns:

Wrapper object if found, else None

Return type:

Optional[gym.Wrapper]
