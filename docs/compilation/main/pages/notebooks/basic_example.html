Basic example

Sinergym adheres to the the standard Farama Gymnasium API. Let’s explore how to create a basic control loop.

To begin, we need to import Sinergym and create an environment. In this example, we will use the Eplus-demo-v1 environment.

[1]:

import gymnasium as gym
import numpy as np

import sinergym

env = gym.make('Eplus-demo-v1')

At first glance, Sinergym might seem to be only imported and not used. However, importing Sinergym inherently defines all the available Environments. In this case, Eplus-demo-v1 is a pre-defined environment that is just ready to use.

With this straightforward setup, we are prepared to iterate over episodes. For this basic example, we will consider a single episode.

[2]:

for i in range(1):
    obs, info = env.reset()
    rewards = []
    truncated = terminated = False
    current_month = 0
    while not (terminated or truncated):
        a = env.action_space.sample()
        obs, reward, terminated, truncated, info = env.step(a)
        rewards.append(reward)
        if info['month'] != current_month:  # display results every month
            current_month = info['month']
            print('Reward: ', sum(rewards), info)
