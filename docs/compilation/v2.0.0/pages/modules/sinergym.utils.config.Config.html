Config object to manage extra configuration in Sinergym experiments.

Parameters:

  • _idf_path ‚Äì IDF path origin for apply extra configuration.

  • _weather_path ‚Äì EPW path origin for apply weather to simulation.

  • _ddy_path ‚Äì DDY path origin for get DesignDays and weather Location

  • experiment_path ‚Äì Path for Sinergym experiment output

  • episode_path ‚Äì Path for Sinergym specific episode (before first simulator reset this param is None)

  • max_ep_store ‚Äì Number of episodes directories will be stored in experiment_path

  • config ‚Äì Dict config with extra configuration which is required to modify IDF model (may be None)

  • _idd ‚Äì IDD opyplus object to set up Epm

  • building ‚Äì opyplus Epm object with IDF model

  • ddy_model ‚Äì opyplus Epm object with DDY model

  • weather_data ‚Äì opyplus WeatherData object with EPW data

  • action_definition ‚Äì Dict with action definition to automatic building model preparation.
