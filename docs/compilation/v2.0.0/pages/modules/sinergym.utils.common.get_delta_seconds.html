sinergym.utils.common.get_delta_seconds

sinergym.utils.common.get_delta_seconds(st_year: int, st_mon: int, st_day: int, end_year: int, end_mon: int, end_day: int) → float

Returns the delta seconds between st year:st mon:st day:0:0:0 and end year:end mon:end day:24:0:0.

Parameters:

  • st_year (int) – Start year.

  • st_mon (int) – Start month.

  • st_day (int) – Start day.

  • end_year (int) – End year.

  • end_mon (int) – End month.

  • end_day (int) – End day.

Returns:

Time difference in seconds.

Return type:

float
