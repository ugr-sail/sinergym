Sinergym utilizes the standard Farama Gymnasium API. Let‚Äôs explore how to create a basic loop.

To begin, we need to import Sinergym and create an environment. In this example, we will use the Eplus-demo-v1 environment.

[2]:

import gymnasium as gym
import numpy as np

import sinergym
env = gym.make('Eplus-demo-v1')

At first glance, Sinergym might seem to be only imported and not used. However, importing Sinergym inherently defines all its Environments for use. In this instance, Eplus-demo-v1 is readily available with all its features.

With this straightforward setup, we‚Äôre prepared to iterate over the episodes. For this basic example, we‚Äôll consider just one episode. Essentially, the required code would look something like this:

[3]:

for i in range(1):
    obs, info = env.reset()
    rewards = []
    truncated = terminated = False
    current_month = 0
    while not (terminated or truncated):
        a = env.action_space.sample()
        obs, reward, terminated, truncated, info = env.step(a)
        rewards.append(reward)
        if info['month'] != current_month:  # display results every month
            current_month = info['month']
            print('Reward: ', sum(rewards), info)
